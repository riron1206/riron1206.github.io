【簡単】GithubでWebサイトを公開する2つの方法

  • git clone https://github.com/riron1206/riron1206.github.io
  • cd riron1206.github.io
  • echo "Hello World" > index.html # https://www.tam-tam.co.jp/tipsnote/html_css/post11245.html のサイトのindex.htmlでpushし直した
  • git add --all # フォルダ内のファイル全push
  • git commit -m "Initial commit" # pushをコミット
  • git push -u origin master # マスターブランチにpush?
